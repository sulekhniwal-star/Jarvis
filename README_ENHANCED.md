### Environment Variables
```env
# Core (Required)
GEMINI_API_KEY=your_key_here

# Information APIs (Optional)
NEWS_API_KEY=your_news_key
WEATHER_API_KEY=your_weather_key

# Communication (Optional)

# Assistant Settings

```

        return "Custom response!"

 Q: "Will I be asked for admin password every time?"
 A: NO! Admin is only needed for the initial setup.
    After that, JARVIS runs automatically with no prompts.

 Q: "Does it work for multiple users?"
 A: YES! When ANY user logs in, JARVIS starts automatically.
    All users share the same memory system.

 Q: "Can I disable it if I don't want it running?"
 A: YES! Easy - just disable the Task Scheduler task.

 Q: "What if JARVIS is already running?"
 A: It won't start twice. The system prevents duplicate instances.

 Q: "Will volume control work?"
 A: Maybe. Windows limits volume changes to admin processes.
    The setup script tries to enable it for you.

 Q: "What if I only want it for one user?"
 A: Delete the public shortcuts and registry entry.
    Or just disable the Task Scheduler task.

 Q: "Does it slow down PC startup?"
 A: Minimal impact (~1-2 seconds added to boot time).

 Q: "Can I move the JARVIS folder after setup?"
 A: NO! The paths are hardcoded in Task Scheduler and registry.
    You would need to run setup again from the new location.
